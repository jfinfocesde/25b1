    Semana 7 - Actividad Práctica: Implementación de Herencia

    Objetivo

    Implementar un sistema de herencia en Java aplicando los conceptos aprendidos en clase, utilizando control de versiones con Git y GitHub.

    Instrucciones

    Paso 1: Configuración del Repositorio

    1. Hacer Fork del Repositorio Base
    2. Accede al repositorio: https://github.com/jfinfocesde/act_b1_s7.git
    3. Haz clic en el botón "Fork" en la esquina superior derecha
    4. Selecciona tu cuenta personal como destino del fork

    5. Espera a que se complete el proceso de fork

    6. Clonar tu Fork Localmente 

      git clone https://github.com/TU_USUARIO/act_b1_s7.git
cd act_b1_s7

    7. Configurar Información Personal

    8. Abre el archivo info.json en la raíz del proyecto
    9. Completa la información solicitada:
    {
    "identificacion": "1234567890",
    "nombres": "Tu Nombre Completo",
    "apellidos": "Tus Apellidos Completos", 
    "grupo": "b1-martes"
}
